# IBM Data Analyst Professional Certificate - Capstone Project
### Leveraging Data Science to Analyze Technology Trends and Market Demand

## 📊 Project Overview
This project serves as the culmination of the **9-course IBM Data Analyst Professional Certificate**. The primary objective was to analyze a comprehensive dataset of developer surveys to identify emerging technology trends, salary distributions, and demographic shifts in the global tech landscape.

As a **Business and Marketing professional**, I approached this data with a focus on actionable insights, moving beyond raw numbers to understand how these trends impact organizational strategy and recruitment.

## 🛠️ Tools & Technologies Used
* **Python:** Data Wrangling (Pandas), Exploratory Data Analysis (Numpy), and Visualization (Matplotlib, Seaborn).
* **SQL:** Data extraction and filtering using SQLite to query large survey databases.
* **Data Visualization:** Created interactive dashboards and a 14-slide professional report.
* **Microsoft Excel:** Initial data exploration and pivot table analysis.

## 📁 Repository Contents
* `Data_Analysis_with_Python.ipynb`: The complete Jupyter Notebook containing code for data cleaning, EDA, and statistical plotting.
* `Database_Queries.sql`: A collection of SQL queries used to extract specific insights from the survey database.
* `Final_Presentation.pdf`: A comprehensive 14-slide report summarizing findings for stakeholders.

## 🔍 Key Findings & Analysis
* **Technology Trends:** Identified the top 5 most-demanded programming languages and their growth trajectory over the next 2 years.
* **Salary Insights:** Conducted a deep dive into salary distributions across different regions and experience levels.
* **Demographic Analysis:** Visualized the relationship between education levels and career satisfaction using boxplots and histograms.
* **Market Demand:** Used 2x2 matrix visualizations to compare current technology usage vs. future interest.
